# VoyaGo
VoyaGo — Bora viajar.

Descrição
-------
VoyaGo é um aplicativo para montar roteiros de viagem automaticamente a partir de destinos, datas e preferências do usuário. Este repositório contém dois subprojetos: o back-end (Java/Spring Boot) responsável por gerar e armazenar roteiros, e o front-end (React Native + Expo) para experiência mobile.

Estrutura do repositório
------------------------
- `Back-End/` — código-fonte do servidor (Spring Boot, JPA, Postgres/H2 para dev).
- `Front-end/` — aplicação mobile em React Native (Expo-managed).

Visão rápida (quickstart)
-------------------------
1. Backend: dentro de `Back-End/` você encontrará instruções específicas no `Back-End/README.md` para configurar e executar o servidor.
2. Frontend: dentro de `Front-end/` siga `Front-end/README.md` para iniciar o app via Expo.

Como contribuir
---------------
- Abra issues para bugs, sugestões de features e melhorias no gerador de roteiros.
- Para contribuições de código, crie uma branch com um nome descritivo, faça commits pequenos e abra um pull request apontando para `main`.
